#### ⚡ LeadTrack AI

Automated Lead Management & Email Tracking System

🔗 **Live Demo:** [leadtrack-ai.vercel.app](https://leadtrack-ai.vercel.app)

---

## What it does

- 📋 Lead capture form with validation
- 🗄️ Stores data in Supabase database
- 📧 Auto sends personalized email on submission
- 👁️ Tracks when email is opened
- 🔗 Tracks when link is clicked
- 📊 Live analytics dashboard
- 🤖 AI classifies lead category & priority

---

## Tech Used

React • Vite • Tailwind CSS • Supabase • EmailJS • Recharts

---

## Quick Start

```bash
git clone https://github.com/YOUR-USERNAME/leadtrack-ai.git
cd leadtrack-ai
npm install
cp .env.example .env.local
npm run dev
```

---

## Environment Variables

```env
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
VITE_EMAILJS_SERVICE_ID=
VITE_EMAILJS_TEMPLATE_ID=
VITE_EMAILJS_PUBLIC_KEY=
VITE_APP_URL=http://localhost:5173
```
